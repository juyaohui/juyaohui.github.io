# 记录一下游戏中最刺激的时刻

都是一些最高光瞬间，也有很多平凡的运营局，但那些不过是公式人大战公式人而已，没有恋恋不舍的感觉

# 破损秒表三星烬

这是刚玩这游戏前几把吃鸡，不是第一把就是第二把。开局拿到了发明家造物（复活衍生物），前面没有过多掉血。第三个海克斯拿到了破损的秒表，围绕发明家的思路转换成为围绕精密类英雄，也就是烬和发条。所以重心放在了提升烬的质量

最后几个回合追到了三星，对面也是狠人，三星刀妹。不过我的海克斯加上烬太猛了，合起来就是看着我打，枪枪致命！

# 暗星变异三星卡莎

S6.5变异类型有七种，好几种都不太会玩，不知道应该以谁为核心，怎么发挥羁绊的优势。有几种比较热门的变异类型，大家都会玩。减蓝变异，核心外援狐狸，中期强无敌，只要打手短阵容都是稳稳拿下。后期回合核心依然是卡莎，不然场景就很尴尬，被远程秒杀。暗星变异，这应该是最热门最强的变异，目前段位感受到的。配合假人海克斯，腰带和攻速那个装备。只要出现这个变异，直接拉人口追逐卡莎，追不到那就下一把。不能惯着其他这样玩的人，如果你不加入这内卷，那到后面的回合你会感受到一个大招毁天灭地的威力，想吃鸡就只能选这个，其他羁绊真的没法玩。

在这把游戏对局中，我早早拿到了马尔扎哈，前期装备都给她，以她为核心，这样才能走到后期。后面升人口拿到卡莎已经非常激动了，因为暗星变异只要三人羁绊就很强。所以我留了找到的卡莎，直接双卡莎，中期一度无敌。后面看了一眼别人的阵容都没有卡莎，那我就追一下三星卡莎。现在版本有龙蛋，大龙蛋会给光明装备。光顾着找卡莎了，我都没看别人在干嘛，对面光明法衣，正对着我所有人。三星德莱文，感觉还放双卡莎的话，对面已经赢了。还好我技高一筹（够狗），直接三星卡莎，被法衣全对着也有惊无险的拿下。

# 三星泽丽烟花秀

在几种大羁绊中，我就喜欢玩狙神，狙神是距离敌人越远伤害越高，这个羁绊伤害倾泻太吓人了。泽丽的攻击距离是九格，这意思就是，泽丽就是狙神中输出最暴力的，两狙神也够。 这局的对手是三星轮子妈，轮子妈手不够长，输出是暴力但是没别的棋子快，在五月底龙蛋版本感觉像是玩具。对面就赢一手艾克跳到我所有人，轮子妈快速启动。经过之前被别人法衣艾克啥的针对，卡着最后几秒把泽丽放在最后，防个机器人钩就行，啥buff也不贪。一启动直接 给对面来了一场烟花秀，视觉满分。

# 三星爆爆

在这个段位，大家酷爱玩赏金，这应该是我玩这个游戏最难的阵容了，因为玩这个阵容最好保持连败，这样后面宝箱中会有大奖（妮蔻、装备、金铲铲、五费卡、大量金币等等），后面的收益太高了，只要将赏金提取出来提升在阵容中，前几名随便拿。这个阵容的难度就在于，前期控制连败（我每次碰见约德尔小矮子，就被迫退出大奖），中期果断收菜（额，每次玩着玩着感觉就止不住血了，想收菜又收不掉，含恨而终），所以精髓就在于判断别人阵容强度，及时调整。感觉这个难度很大，大部分人看命收菜，命不好就下一把。

在这局游戏中，其他人都在贪，一看这个情况，直接拉九。前面想着玩挑战德莱文，后面卡莎到了，那就挑战卡莎。后面金克斯到了，一点羁绊也没，我想着那就挂个二星爆爆吧，然后又来了一个二星爆爆，两个爆爆没啥羁绊也不强，打不过别人。这个时候看一眼，别人都没有拿金克斯，选秀给反甲金克斯，其余人都嫌弃，那我就追一下。成功追出三星，感觉美滋滋，最近这段时间很火的露露七仙女阵容，在金克斯的大招面前就像以前齐天大圣电视剧里那个听谛一样，一脚给它踏碎了哈哈哈。

那个八保镖阵容就是玩具，怎么打都打不过我。被刺转塔姆加螳螂那小子阴了一手，被他的法衣照了，螳螂塔姆跳脸上还没启动就没了。好在我血多，没成为死不瞑目。抱团沉底防刺客稳得，螳螂打不过八保镖这把就结束了。

# 璀璨钻石！

2022.06.06

今天是六月六号，从五月五号算起，刚好过了一个月。前几天觉得赛季快要结束了，便想上个钻石来看看自己的实力。一旦定下了目标，剩下的就是全力以赴。昨天的心情无比沮丧，我数了数，一天打了十九把游戏，一天中超过一半的时间都在打，打到头脑混乱，打到眼神迷离，却还是停留在铂金二。今天端午假期结束了，原本想放弃这个目标，但是学习上依然没有什么好消息，还是再试一试吧。起初也还是起起落落，但想着一直到晚上十一点，还有很多时间，就再认真上分。上铂金一之后，心情变得亢奋起来，分奴嘴脸尽显无疑，只玩一套阵容，迪加狙以及类似阵容，这样做是有效果的。装备都烂熟于心，英雄也大差不差。想要的东西都很明确，这游戏就非常简单。

下面是战绩图

每个分段快要接近上个分段的人是最认真的。这比我在铂金四遇到的那些人厉害不少，之前的阵容现在再用未必能吃鸡。最后一把四狙四精密三白魔，破损的秒表对烬太好用了，刀妹这种根本启动不了。

最后是快乐时刻！
